## 配色ルール

### カラーパレット

| トークン | HEX | RGB | 役割 |
|---------|------|-----|------|
| **FEATHER** | `#77C9D4` | 119, 201, 212 | ライトアクセント・背景グラデーション（明側）・ホバー |
| **MARINE** | `#57BC90` | 87, 188, 144 | プライマリカラー・ボタン・ヘッダー・リンク |
| **FOREST** | `#015249` | 1, 82, 73 | ダークアクセント・見出し強調・フッター背景 |
| **SLEEK GREY** | `#A5A5AF` | 165, 165, 175 | サブテキスト・ボーダー・無彩色バランス |

### 基本ルール

- **背景:** 白 `#FFFFFF` をベースとし、セクション交互背景には `#F7FAFA`（FEATHER の極薄版）を使用
- **テキスト:** 本文は `#1A1A1A`、サブテキストは SLEEK GREY `#A5A5AF`
- **CTA ボタン:** MARINE `#57BC90` を背景、白文字。ホバー時は FOREST `#015249` に遷移
- **グラデーション:** FEATHER → MARINE の左→右（ヒーローセクション等）
- **フッター:** FOREST `#015249` 背景、テキストは `#D1E8E4`（FEATHER の淡色）
- **ボーダー・区切り線:** SLEEK GREY `#A5A5AF` の 30% 透過（`rgba(165, 165, 175, 0.3)`）

### CSS 変数定義

```css
:root {
  --color-feather: #77C9D4;
  --color-marine: #57BC90;
  --color-forest: #015249;
  --color-sleek-grey: #A5A5AF;

  --color-bg-primary: #FFFFFF;
  --color-bg-section-alt: #F7FAFA;
  --color-text-primary: #1A1A1A;
  --color-text-secondary: #A5A5AF;
  --color-footer-bg: #015249;
  --color-footer-text: #D1E8E4;
  --color-border: rgba(165, 165, 175, 0.3);
}
```

### 禁止事項

- パレット外の色を独自に追加しないこと（必要な場合はこのドキュメントを更新してから使用）
- MARINE と FEATHER を隣接させる際は、十分なコントラスト比（4.5:1 以上）を白背景で確保すること
- FOREST を大面積の本文テキストに使用しないこと（見出し・強調のみ）

### 模倣すべき設計パターン

1. **検索ファースト:** トップページの最上部に地域検索を配置。ユーザーが即座にアクションできる
2. **数値で信頼構築:** 「10,452件」「100万件」「51,838件」など具体的な数値を大きく表示
3. **タイプ別タブ:** 樹木葬/納骨堂/永代供養墓/一般墓をタブで切替え。1画面で全タイプ閲覧可能
4. **口コミ・評価の可視化:** ★レーティング + 口コミ件数を全霊園カードに表示
5. **価格帯の明示:** 区画タイプ別の最低価格を「〜万円〜」形式で表示
6. **GPS連携:** 「現在地から近くの霊園を探す」機能
7. **グループサイト導線:** フッターに関連サービス群（葬儀/仏壇/相続/介護/散骨）を網羅的に配置
