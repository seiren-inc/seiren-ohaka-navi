# お墓探しナビ SEOマスタープラン（神奈川県制圧用）

## 1. キーワード戦略
本サイト（お墓探しナビ）は、神奈川県内における「新規のお墓探し」に関する検索体験を独占することを目標とする。
「改葬（お墓じまい・引越し）」に関連するキーワードは意図的に除外し、別ブランド（お墓じまいナビ）で拾う設計とする。

- **対象エリア**: 神奈川県内 33市町村
- **重点ターゲットキーワード**:
  - `[自治体名] 永代供養`
  - `[自治体名] 樹木葬`
  - `[自治体名] 納骨堂`
  - `[自治体名] 霊園 比較`
- **除外・ネガティブキーワード**: 
  - `改葬`, `墓じまい`, `引越し`, `手続き`
  - （※該当検索意図を持つユーザーにはBridgeSection経由で別サイトへ送客）

---

## 2. URL階層設計
検索エンジンおよびAIがサイト構造を正確にクロール・理解できるよう、ディレクトリ階層を以下のように県・市町村・施設種別の順で物理的・論理的にマッピングする。

- **県トップ**: `/kanagawa/`
- **市町村トップ**: `/kanagawa/{municipality}/`
- **市町村別・永代供養**: `/kanagawa/{municipality}/eitaikuyo`
- **市町村別・樹木葬**: `/kanagawa/{municipality}/jyumokuso`
- **市町村別・納骨堂**: `/kanagawa/{municipality}/noukotsudo`

※ `{municipality}` には横浜市 (`yokohama-shi`) などのローマ字表記を充てる。

## 3. title / description テンプレート
クリック率（CTR）を高め、検索意図に直結するフォーマットを全ページで固定化する（Next.js `generateMetadata` で自動生成）。

### 市町村トップ（例：横浜市）
- **title**: `横浜市の霊園・お墓一覧｜永代供養・樹木葬の費用相場と見学予約【お墓探しナビ】`

## 4. noindex 設計
クローラビリティを担保するため、検索エンジンにインデックスさせる価値のない以下のページ群には `<meta name="robots" content="noindex" />` を確実に出力する。

1. **管理画面**: `/admin` 配下の全ページ
2. **パラメータ付きURL**: UI制御のみに使われる `?sort=`, `?filter=`, `?page=` などのURL（Canonicalで正規化またはnoindex）
3. **重複・低品質**: コンテンツが全くない空の自治体ページや、自動生成されただけの中身のないページ
4. **内部システム用**: `/api` 直接アクセス、完了画面（`/consult/grave-search?success=true` 等）

## 5. 構造化データ（JSON-LD）設計
検索結果画面でのリッチリザルト表示を獲得し、CTRを最大化する。

- **施設詳細ページ**:
  - `LocalBusiness` （または `Cemetery`, `Place`）: 施設名、住所、電話番号、営業時間、価格帯などを定義。
  - `FAQPage`: 当該施設に関するよくある質問（管理費、宗教宗派など）。
- **ガイド系・コラムページ**:
  - `Article`: 執筆者情報、公開日時、更新日時。
  - `FAQPage`: 記事内にあるQ&Aセクションを抽出。
- **エリア・一覧ページ**:
  - `BreadcrumbList`: `トップ > 神奈川県 > 横浜市 > 樹木葬` のようなパンくず階層。
  - `ItemList` (Carousel仕様): リスト表示される霊園一覧を定義。

## 6. AI引用最適文章構造（SGE・ChatGPT検索対策）
ゼロクリック検索やAIによるオーバービュー（SGE）での引用率を上げるため、ページ内の主要セクション（特に概要や見解部分）においては、以下の文章構造を徹底する。

1. **結論 (Conclusion)**: 最も重要な答えを冒頭1段落で簡潔に述べる。
2. **理由 (Reason)**: その結論に至る根拠や背景を1〜2段落で説明する。
3. **箇条書き (Bullet points)**: 詳細な条件や特徴を、構造的に（AIが抽出しやすい形で）リスト化する。

**📝 実装イメージ（横浜市の樹木葬費用相場の場合）:**
> **【結論】** 横浜市における樹木葬の平均的な費用相場は、約40万円〜80万円です。 
> **【理由】** 地価が比較的高いエリアであるため県内の平均より若干高めですが、個別区画の専有期間（13年・33年など）やプレートの装飾度合いにより価格が大きく変動します。
> **【費用が決まる主な要因（箇条書き）】**
> - 個別区画の有無（合祀か個別か）
> - 納骨できる人数（1名・夫婦・家族）
> - 年間管理費の有無と金額
> - 墓碑（プレート）の材質や彫刻費用
