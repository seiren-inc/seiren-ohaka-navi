# 橋渡しコンポーネント (BridgeSection) 実装仕様

## 1. 目的
「お墓探しナビ」のサイトテーマ純度（新規のお墓・永代供養・樹木葬・納骨堂）を維持しつつ、「現在のお墓をどうにかしたい（改葬・墓じまい）」という検索意図を持つユーザーを、専門ブランドである「お墓じまいナビ」へ自然に送客する。

---

## 2. コンポーネント基本仕様
- **ファイルパス**: `components/BridgeSection.tsx`（または適切なディレクトリ）
- **Props**:
  - `municipalityName` (string): 自治体名（例："横浜市", "川崎市"）
  - `bridgeUrl` (string): 送客先となるお墓じまいナビの該当自治体ページURL

---

## 3. 表示位置・表示条件
- **施設詳細ページ**: 「費用」「供養内容」セクションの直後に配置。
- **市区町村ページ**: ページ下部の「FAQ（よくある質問）」セクションの直前に配置。
- **制限事項**: 1ページにつき最大 **1回のみ** の表示とする（テーマのノイズにならないようにするため）。

---

## 4. UI / UX 仕様
目立たせすぎず、しかし必要なユーザーの目には留まる「情報ボックス」としてのデザインを採用する。

- **外観**: 薄い背景色（またはグレー系）の控えめな枠付きボックス。
- **見出し (H3)**:
  - テキスト: `改葬を検討中の方へ`
  - ※ サイトの構造化ルール（H2の乱用防止）に従い、必ず `<h3>` を使用すること。
- **本文**:
  - `現在のお墓を整理し、こちらへ改葬される場合は、改葬許可証の取得や行政手続きが必要です。{municipalityName}の手続き方法は専門サイトで解説しています。`
- **ボタン**:
  - テキスト: `{municipalityName}の改葬許可証取得方法を見る`
  - デザイン: プライマリカラーの過度なベタ塗りを避け、アウトラインボタン（または控えめな配色のボタン）とする。
- **リンク属性**:
  - 内部ブランド間のクローラビリティとLink Equityの受け渡しを考慮し、**`rel="nofollow"` は付与しない**。
  - 自然なページ遷移を促すため **`target="_blank"` は付与しない**（同一タブで遷移させる）。

---

## 5. Google Analytics 計測仕様
送客効果を測定するため、ボタンクリック時にGAイベントを発火させる。

- **イベント名**: `bridge_click`
- **送信パラメータ (Event Parameters)**:
  - `municipality`: `{municipalityName}` （対象自治体名）
  - `pageType`: `"facility"`（施設詳細ページの場合）または `"municipality"`（市区町村ページの場合）
